Распознавание и проверка математических примеров по фото 📸🧮
Программа для автоматического распознавания математических выражений с фотографий и проверки правильности решения.

Что делает программа  (Инструкция в папке)
Считывает текст с изображения с помощью EasyOCR (поддержка русского языка)

Распознает математический пример вида выражение = ответ

Вычисляет результат с помощью библиотеки numexpr

Сравнивает с распознанным ответом

Выставляет оценку по 5-балльной шкале

Возможности
📷 Поддержка фото с рукописными/печатными примерами

🔄 Автоматическая замена похожих символов (X → *, : → / и т.д.)

🧮 Поддержка операций: +, -, *, /, (, )

📊 Вывод результата проверки и оценки

⚠️ Обработка ошибок при недопустимых символах

Пример работы
text
Фото: example.jpg
Ваш пример: 2+2*2=6
Правильный пример: 2+2*2 = 6.0
Правильно решено примеров: 1/1, 100% - оценка 5
Шкала оценок
Процент	Оценка
≥ 80%	5
65–79%	4
50–64%	3
< 50%	2
Технологии
EasyOCR — распознавание текста с изображений

NumExpr — безопасное вычисление математических выражений

⚠️ Примечание: текущая версия обрабатывает только один пример за раз (распознает первое найденное выражение).
